# Therapeutic Mechanism of Zhenwu Huangqi Decoction on Chronic Heart Failure with Reduced Ejection Fraction

## About 
This repository contains fully reproducible code and complete research documentation for investigating the therapeutic mechanism of Zhenwu Huangqi Decoction (ZWHQD) in the treatment of heart failure with reduced ejection fraction (HFrEF).
